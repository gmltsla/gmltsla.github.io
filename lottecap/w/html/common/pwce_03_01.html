공인인증서 내보내기

공인인증서가 지정된 PC에 별도의 연결장치 없이 공인인증서를 고객님의 스마트폰으로 전송 할 수 있습니다.

공인인증서 내보내기

PC의 공인인증서를 스마트폰으로 내보내기 진행 시 스마트폰 내 롯데캐피탈 앱을 실행 하셔야 합니다.

인증서 중계 프로그램 설치
공인인증서 복사

인증서 중계 프로그램 설치를 완료 하셔야 공인인증서 복사가 가능합니다.

공인인증서 내보내기 절차

  • 1단계 공인인증서 복사 버튼 클릭
  • 2단계 공인인증서 비밀번호 입력
  • 3단계 롯데캐피탈 앱 > 공인인증센터
  • 4단계 인증서 가져오기 선택
  • 5단계 PC 8자리 숫자 확인
  • 6단계 인증서 가져오기 하단 입력영역 (8자리 숫자입력)
  • 7단계 공인인증서 비밀번호 입력
  • 8단계 공인인증서 가져오기 완료
